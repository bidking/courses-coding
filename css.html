CSS Tutorial
Pengenalan CSS

Cascading Style Sheets (CSS) adalah bahasa stylesheet yang digunakan untuk mengatur tampilan elemen-elemen dalam sebuah dokumen HTML. CSS memungkinkan Anda untuk mengontrol warna, font, ukuran, jarak, dan tata letak dari elemen-elemen halaman web.

Beberapa konsep utama dalam CSS meliputi:

  • Selector: CSS menggunakan selector untuk menargetkan elemen HTML dan menerapkan gaya padanya.
  • Properti: Properti CSS mendefinisikan tampilan elemen HTML, seperti warna, ukuran font, lebar, tinggi, dll.
  • Nilai: Properti CSS diberikan nilai yang menentukan bagaimana pengaruhnya terhadap tampilan elemen.
  • Kelas dan ID: Kelas dan ID CSS digunakan untuk menerapkan gaya pada elemen tertentu atau grup elemen.
  • Model Kotak: Model kotak CSS menjelaskan bagaimana elemen dirender di halaman, termasuk konten, padding, border, dan margin.
  • Tata Letak: CSS dapat digunakan untuk membuat struktur tata letak yang berbeda, seperti tata letak grid, tata letak flexbox, dan tata letak berbasis float.
Selector CSS

Selector CSS digunakan untuk menentukan elemen mana yang akan diberikan gaya. Berikut adalah beberapa contoh selector CSS yang umum digunakan:

  • Selector Elemen: Selector elemen langsung menargetkan elemen HTML, seperti p untuk paragraf atau h1 untuk judul level 1.
  • Selector Kelas: Selector kelas digunakan untuk menargetkan elemen dengan kelas tertentu, seperti .container.
  • Selector ID: Selector ID menargetkan elemen dengan ID tertentu, seperti #header.
  • Selector Universal: Selector universal (*) menargetkan semua elemen dalam dokumen.
  • Selector Pseudoclass: Selector pseudoclass digunakan untuk menargetkan elemen dalam keadaan tertentu, seperti :hover untuk saat elemen sedang dihover oleh kursor.
Properti CSS

Properti CSS digunakan untuk mengontrol tampilan dan perilaku elemen HTML. Berikut adalah beberapa contoh properti CSS yang umum digunakan:

  • color: Mengatur warna teks.
  • font-size: Mengatur ukuran font.
  • background-color: Mengatur warna latar belakang elemen.
  • padding: Mengatur ruang di dalam batas elemen.
  • margin: Mengatur ruang di luar batas elemen.
  • border: Mengatur garis batas elemen.
  • display: Mengatur cara elemen dirender di halaman.
  • position: Mengatur posisi elemen dalam tata letak dokumen.
  • float: Mengatur apakah elemen mengapung di sebelah kiri atau kanan dari elemen sekitarnya.
Tata Letak CSS

Tata letak CSS mengacu pada cara elemen-elemen dalam halaman web diatur dan diorganisasi. Berikut adalah beberapa teknik tata letak CSS yang umum digunakan:

  • Grid Layout: Grid layout memungkinkan Anda untuk membuat tata letak yang kompleks dengan menggunakan baris dan kolom.
  • Flexbox Layout: Flexbox layout memberikan cara yang lebih mudah untuk mengatur elemen dalam baris atau kolom.
  • Float Layout: Float layout mengapungkan elemen di sebelah kiri atau kanan elemen lain.
  • Positioning: Positioning memungkinkan Anda untuk menentukan posisi elemen secara tepat dalam tata letak.
