JavaScript Tutorial
Pengenalan JavaScript

JavaScript adalah bahasa pemrograman yang sering digunakan di web untuk menambahkan interaktivitas pada halaman web. Dengan JavaScript, Anda dapat mengontrol perilaku halaman web, mengubah konten HTML dan CSS secara dinamis, serta berinteraksi dengan pengguna melalui formulir, tombol, dan elemen lainnya.

Beberapa konsep utama dalam JavaScript meliputi:

  • Variabel: Variabel digunakan untuk menyimpan nilai dan dapat diubah selama eksekusi program.
  • Tipe Data: JavaScript memiliki beberapa tipe data, termasuk string, number, boolean, array, dan object.
  • Operator: Operator digunakan untuk melakukan operasi pada variabel dan nilai.
  • Kondisional: Kondisional, seperti if-else statement, digunakan untuk menjalankan blok kode berdasarkan kondisi tertentu.
  • Loop: Loop, seperti for dan while loop, digunakan untuk mengeksekusi blok kode secara berulang.
  • Fungsi: Fungsi digunakan untuk mengelompokkan blok kode yang dapat dipanggil dan digunakan kembali.
  • Event Handling: JavaScript digunakan untuk menangani event, seperti klik mouse atau input pengguna.
  • DOM Manipulation: Document Object Model (DOM) memungkinkan JavaScript untuk mengubah struktur, gaya, dan konten halaman web.
Variabel JavaScript

Variabel dalam JavaScript digunakan untuk menyimpan data dan memberi nama pada data tersebut agar dapat diakses dan digunakan kembali dalam program. Berikut adalah beberapa hal yang perlu diketahui tentang variabel dalam JavaScript:

  • Deklarasi: Variabel dideklarasikan menggunakan kata kunci var, let, atau const.
  • Tipe Data: JavaScript memiliki tipe data dinamis, yang berarti Anda tidak perlu menentukan tipe data secara eksplisit saat mendeklarasikan variabel.
  • Scope: Variabel JavaScript memiliki scope, yang mengatur di mana variabel dapat diakses dalam program.
  • Hoisting: Variabel JavaScript dapat "ditinggikan" (hoisted) ke atas cakupan fungsinya, yang berarti Anda dapat menggunakan variabel sebelum dideklarasikan.
  • Nilai Awal: Jika variabel tidak diinisialisasi, maka nilainya akan undefined.
Fungsi JavaScript

Fungsi dalam JavaScript digunakan untuk mengelompokkan blok kode yang dapat dipanggil dan digunakan kembali. Berikut adalah beberapa hal yang perlu diketahui tentang fungsi dalam JavaScript:

  • Deklarasi: Fungsi dapat dideklarasikan menggunakan kata kunci function.
  • Parameter: Fungsi dapat menerima parameter, yang digunakan untuk mengirim data ke dalam fungsi.
  • Pengembalian Nilai: Fungsi dapat mengembalikan nilai menggunakan kata kunci return.
  • Ekspresi Fungsi: Fungsi juga dapat didefinisikan menggunakan ekspresi fungsi atau fungsi anonim.
  • Arrow Function: Arrow function adalah cara baru untuk mendeklarasikan fungsi dalam JavaScript yang lebih singkat.
  • Callback: Callback adalah fungsi yang dikirimkan sebagai argumen ke fungsi lain dan akan dipanggil setelah fungsi tersebut selesai dieksekusi.
Manipulasi DOM JavaScript

Manipulasi Document Object Model (DOM) adalah proses mengubah struktur, gaya, dan konten halaman web menggunakan JavaScript. Berikut adalah beberapa cara untuk memanipulasi DOM dalam JavaScript:

  • Seleksi Elemen: JavaScript dapat digunakan untuk memilih elemen-elemen di halaman web menggunakan metode seperti document.getElementById() atau document.querySelector().
  • Manipulasi Konten: Anda dapat mengubah konten elemen, seperti teks atau atribut, menggunakan properti seperti innerHTML atau textContent.
  • Manipulasi Gaya: Anda dapat mengubah gaya elemen menggunakan properti seperti style atau classList.
  • Menambahkan atau Menghapus Elemen: JavaScript dapat digunakan untuk menambahkan atau menghapus elemen dari halaman web menggunakan metode seperti appendChild() atau removeChild().
  • Event Handling: JavaScript dapat digunakan untuk menangani event, seperti klik mouse atau input pengguna, dan menanggapi event tersebut dengan menjalankan fungsi tertentu.
