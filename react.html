ReactJS Tutorial
Pengenalan React

ReactJS adalah pustaka JavaScript yang digunakan untuk membangun antarmuka pengguna (UI) dalam aplikasi web. React memungkinkan pengembang untuk membuat komponen UI yang dinamis dan reaktif, yang memungkinkan aplikasi untuk merespons perubahan data dengan cepat dan efisien. Beberapa konsep utama dalam ReactJS meliputi:

  • Komponen: React memecah UI menjadi komponen-komponen yang dapat dikelola secara terpisah.
  • Props: Props (atau properties) digunakan untuk mengirim data dari komponen induk ke komponen anak.
  • State: State digunakan untuk menyimpan data yang akan berubah selama siklus hidup komponen.
  • Virtual DOM: React menggunakan struktur data yang disebut Virtual DOM untuk memperbarui UI secara efisien.
  • Cycle of Life: Komponen React memiliki siklus hidup yang terdiri dari beberapa metode yang dipanggil dalam urutan tertentu.
  • Event Handling: React memungkinkan penanganan event menggunakan sintaks yang serupa dengan HTML.
Komponen React

Komponen adalah bagian penting dari pengembangan aplikasi React. Mereka memungkinkan Anda untuk membagi UI menjadi bagian-bagian yang lebih kecil, yang dapat dikelola secara terpisah. Berikut adalah beberapa jenis komponen yang umum digunakan dalam React:

  • Komponen Fungsional: Komponen fungsional adalah fungsi JavaScript biasa yang menerima props sebagai parameter dan mengembalikan elemen React.
  • Komponen Kelas: Komponen kelas adalah kelas JavaScript yang mewarisi dari kelas React.Component dan memiliki state serta metode siklus hidup.
  • Komponen Anak: Komponen anak adalah komponen yang ditempatkan di dalam komponen lain dan menerima data melalui props.
  • Komponen Tertanam: Komponen tertanam adalah komponen yang diimpor dan digunakan di dalam komponen lain.
  • Komponen Presentasi dan Kontainer: Komponen presentasi (dumb components) hanya bertanggung jawab untuk menampilkan UI, sementara komponen kontainer (smart components) mengelola logika aplikasi dan state.
Props React

Props (atau properties) adalah mekanisme untuk mengirim data dari satu komponen ke komponen lain dalam React. Props bersifat read-only, yang berarti bahwa komponen tidak dapat mengubah props yang diterimanya. Berikut adalah beberapa hal yang perlu diketahui tentang props dalam React:

  • Penggunaan: Props digunakan untuk mengirim data dari komponen induk ke komponen anak.
  • Read-only: Props bersifat read-only, sehingga komponen tidak dapat mengubah props yang diterimanya.
  • Deklarasi: Props dideklarasikan sebagai atribut pada elemen komponen dalam sintaks JSX.
  • Destructuring: Anda dapat menggunakan destructuring untuk mengakses nilai props dengan lebih mudah.
  • Default Props: Anda dapat menentukan default props untuk komponen menggunakan properti defaultProps.
State React

State digunakan dalam React untuk menyimpan data yang akan berubah selama siklus hidup komponen. State berbeda dengan props dalam hal bahwa state dikelola di dalam komponen, sedangkan props dikirim dari komponen induk ke komponen anak. Berikut adalah beberapa hal yang perlu diketahui tentang state dalam React:

  • Deklarasi: State dideklarasikan dalam konstruktor komponen menggunakan this.state.
  • Perubahan State: State harus diperbarui dengan menggunakan metode this.setState(), yang memicu pembaruan UI.
  • Asynchronous: this.setState() adalah asinkron, jadi Anda tidak dapat mengandalkan nilai state yang baru segera setelah memanggilnya.
  • Stateful Components: Komponen yang memiliki state disebut komponen berkelanjutan (stateful components).
  • Fungsi setState: Anda juga dapat menggunakan fungsi dalam setState() untuk melakukan perubahan berdasarkan nilai state sebelumnya.
Siklus Hidup React

Siklus hidup komponen adalah serangkaian metode yang dipanggil dalam urutan tertentu selama fase kehidupan komponen dalam React. Pemahaman siklus hidup sangat penting untuk mengelola state, melakukan tugas-tugas tertentu pada saat yang tepat, dan mencegah masalah kinerja. Berikut adalah beberapa fase siklus hidup utama dalam React:

  • Mounting: Fase pertama dalam siklus hidup, terjadi saat komponen dipasang pertama kali ke DOM.
  • Updating: Terjadi saat komponen diperbarui dan menerima props baru atau mengalami perubahan state.
  • Unmounting: Fase terakhir dalam siklus hidup, terjadi saat komponen dihapus dari DOM.
  • Error Handling: Fase tambahan yang digunakan untuk menangani kesalahan yang terjadi selama rendering, pembaruan, atau unmounting komponen.
React Hooks

Hooks adalah fitur baru dalam React 16.8 yang memungkinkan Anda menggunakan state dan fungsi lainnya tanpa perlu menulis kelas. Hooks memungkinkan Anda untuk menggunakan fitur-fitur React dalam komponen fungsional. Beberapa hooks yang umum digunakan dalam React adalah:

  • useState: Digunakan untuk menambahkan state ke komponen fungsional.
  • useEffect: Digunakan untuk menangani efek samping dalam komponen fungsional, seperti melakukan panggilan HTTP atau langganan.
  • useContext: Digunakan untuk mengakses konteks dalam komponen fungsional.
  • useReducer: Alternatif untuk useState yang lebih kuat, digunakan untuk mengelola state kompleks.
  • useRef: Digunakan untuk membuat referensi ke elemen DOM atau nilai dalam komponen fungsional.
